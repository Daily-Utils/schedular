# Nest Server

A nestjs server designed for scheduling

## ENV

See the .env.example

Setup the env correctly

## Devlopment database

Check the dev folder on local

```sh
docker compose up -d
```

## How to run?

### Using docker

```sh
docker compose build
docker compose up -d
```

### Run locally for development

Make sure local development database is on

```
yarn run start
```
